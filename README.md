# ParkLao Privacy Policy

## Camera Usage
ParkLao uses the device camera to scan QR codes for parking check-in and check-out. No images or videos are stored.

## Data Collection
We collect minimal data such as phone number and usage activity for app functionality.

## Account Deletion
To delete your account:

1. Open ParkLao app
2. Go to Profile or Settings
3. Select "Delete Account"

Or contact us:

Email: support@parklao.com

## Data Handling
- Data is encrypted in transit
- User data can be deleted upon request
